Edubank is an android smart application for university students. The app can manage university timetables, manage modules, calculate university GPA, and manage exam schedules. Students can get reminders of their classes on time in order that student can attend classes on time. It stores all the details of the module even the lecture hall and instructorâ€™s name. Students can easily get details of the next lecturer without a hassle. The app can calculate the GPA of modules. In order that students can set a goal to achieve more marks to increase the GPA. Another important feature is exam schedules. All the newly added schedules will be displayed at the bottom of the interface and the schedules will be organized according to their ID. All the schedules will have a unique ID.

This application contains university timetables, module details, a GPA calculator, and exam schedules.
